# Enhanced-Console-Calculator
Enhanced Java Calculator

📌 Project Description

This is a console-based Java calculator developed as part of a Java Development Internship task.
The program performs basic mathematical operations and includes a simple temperature unit conversion feature.

🚀 Features

- Addition
- Subtraction
- Multiplication
- Division
- Square Root
- Power Calculation
- Temperature Conversion
  - Celsius to Fahrenheit
  - Fahrenheit to Celsius
- User-friendly console menu

🛠 Technologies Used

- Java
- Scanner Class
- Math Library

📂 Project Structure

EnhancedCalculator.java
README.md

▶ How to Run

1. Compile the program

javac EnhancedCalculator.java

2. Run the program

java EnhancedCalculator
